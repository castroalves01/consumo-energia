# ⚡ Calculadora de Consumo de Energia ⚡

## 📌 Objetivo do sistema
Calcular o consumo mensal de energia de um aparelho.

## 🛠️ Linguagem utilizada
Python

## 📝 Fórmula
consumoMensal = (potencia * horasDia * 30) / 1000

## ⌨️ Como usar
Execute:
py app.py

## 💡 Funções
- Calcula consumo mensal
- Mostra custo estimado
